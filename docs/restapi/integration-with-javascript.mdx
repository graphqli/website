## Using `Axios`
```javascript
const token = 'apito-api-secret';
axios.get('https://api.apito.io/secured/rest/project', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
})
.then((res) => {
  console.log(res.data)
})
.catch((error) => {
  console.error(error)
})
```

## Using `Fetch`
```javascript
const token = 'apito-api-secret';
fetch('https://api.apito.io/secured/rest/project', {
  headers: {
    Authorization: `Bearer ${token}`
  }
}).then(res => res.json())
  .then(json => console.log(json));
```
